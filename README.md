# NeuroFit

**Your AI-Powered Health Companion**

NeuroFit is a personal health agent that intelligently adapts to your lifestyle using wearable data (Apple HealthKit / Google Fit), machine learning, and real-time feedback. It helps users achieve wellness goals across fitness, nutrition, sleep, and mental health — through personalized, actionable suggestions.

---

## 🚀 Features

- 🔗 Connects to Apple HealthKit or Google Fit
- 🧠 Uses Reinforcement Learning to optimize health actions
- 🧾 Onboarding with goals, preferences, and equipment access
- 💪 Build custom workout routines (sets, reps, rest)
- 🍽️ Plan and track personalized food and calorie targets
- 💤 Sleep and activity insights
- 🔔 Smart nudges based on your preferred tone and frequency
- 🧩 RAG-based reasoning from health science sources

---

## 📲 Onboarding Questions

Users answer a brief set of questions on:
- Fitness level & goals
- Workout and nutrition preferences
- Activity level and equipment access
- Notification frequency and preferred coaching style

These answers guide how NeuroFit's AI agent tailors its suggestions.

---

## 🧠 Tech Stack

- **Flutter** – cross-platform mobile app
- **Firebase** – user auth, health data, goal tracking
- **SQL** – for detailed logs, analytics
- **Riverpod** – state management
- **RAG + LLMs** – for health advice, powered by trusted sources
- **Reinforcement Learning** – adaptive recommendation engine
